# Nani-Me

Release host for the **Nani-me** anime apps — hard-subs only, on-device scraping, no accounts.

Grab the latest APKs from the [**Releases**](https://github.com/Syha-01/Nani-Me/releases) page.

| Edition | Description |
| --- | --- |
| Mobile | Phone app (Android). Offline downloads, playback speed 0.5×–2×. |
| TV | Android TV app. D-pad UI, skip intro. |

> Sideloaded APKs: when installing, allow "Install unknown apps" for your browser or file manager. Android's "this file may harm your device" prompt is normal for any APK downloaded outside the Play Store.
